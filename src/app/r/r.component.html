Why R?

R is an open source statistical programming language primarily used in data science and analytics. Industries that use R are finance, banking, healthcare, technology, and many more. Because of the rise of data and analytics over the past 20 years, R has become one of the most popular languages for statistical analysis. Similar to Python, R has many user made packages for improving the analytical capabilities and efficiency of model development and visualization. We can see based on the popularity trend of R that it has been on the rise, and we can likely expect this continue, in part because R is a relatively user-friendly language and there is ample well formatted documentation available on the web. It should be kept in mind that R and Python overlap in many ways, though Python is a more general purpose language, and R is built by statisticians for the purpose of data science.
